# Demo dApp for CallPermit Precompile


## Getting started

Clone the repo:

```
git clone https://github.com/albertov19/callpermit-demo
cd callpermit-demo
```

Install dependencies:

```
yarn
```

Start dev server (localhost:3000 by default):

```
yarn dev
```

Start production server (localhost:3000 by default):

```
yarn start
```
